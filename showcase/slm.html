How it works (internals)

Browser / Frontend → HTTP POST /api/chat → Ollama :11434 → loads → qwen2.5:0.5b → streams NDJSON → onToken()

Docker Compose (deployment)

  ollama:
    image: ollama/ollama:latest
    environment:
      - OLLAMA_ORIGINS=*          # let the browser call the API
    ports:
      - "11434:11434"
    volumes:
      - ollama-data:/root/.ollama # cache the downloaded weights

  ollama-pull:                    # one-shot: pull the model, then exit
    image: ollama/ollama:latest
    depends_on: [ollama]
    environment:
      - OLLAMA_HOST=ollama:11434
    entrypoint: ["/bin/sh", "-c", "sleep 5 && ollama pull qwen2.5:0.5b"]
    restart: "no"
